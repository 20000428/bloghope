# ezBookkeeping
<!-- <Share /> -->
<!-- more -->
启动服务Linux / macOS
```bash
 ./ezbookkeeping server run
```
窗户
> .\ezbookkeeping.exe server run
默认情况下，ezBookkeeping 监听的是端口 8080。然后你可以访问。http://{YOUR_HOST_ADDRESS}:8080/
