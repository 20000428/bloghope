# SAP RANGES

RANGES <变量名> FOR <变量类型>
是自带表头的内表

| 名称 | 参考值 | 描述 |
| --------- | ---- | ---- |
| `SIGN` | I或者E | I-包括;E-排除 |
| `OPTION` | EQ;NE;CP;NP;GE;LT;LE;GT | EQ-等于;NE-不等于;NP-排除参考格式;LT-小于;LT-小于或者等于;CP-包含;NP-不包含;GT-大于;GE-大于或者等于;BT-范围;NB-排除范围 |
| `LOW` | | 描述 |
| `HIGH` |  | 描述 |
