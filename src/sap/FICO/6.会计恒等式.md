## 会计恒等式
<!-- :::tip
::: -->
<!-- 摘要截止标签 -->
资产 = 负债 + 所有者权益 + (收入 - 费用)
<!-- more -->

口诀：
• 资产、成本、费用：增加记借方，正常余额在借方
• 负债、权益、收入：增加记贷方，正常余额在贷方
收入和费用都属于损益类科目。 损益类科目的核心特征是"期间性"和"归零性"
成本是为特定目的（产品、项目、订单）归集的费用，它在财务报表中的身份是动态变化的：
成本类科目的借方余额 = 未完工/未销售的成本 = 资产
成本是"未费用的资产"或"已资产化的费用"，它最终归宿只有两种：
1. 留在资产端（资本化：形成存货、固定资产、无形资产）
2. 进入费用端（费用化：影响当期利润）

所以恒等式无需单独列示成本，它已通过资产和费用两个要素完整表达。
成本是"钱花在资产上的痕迹"，不是"负债或权益的化身"：
• 成本进入资产：说明钱还没"牺牲"（产品未卖）
• 成本进入费用：说明钱已"牺牲"（转化为收入代价）
负债和权益是"资金来源"，成本是"资金去向"，两者是因果关系，不是转化关系。
资产 → 成本 → 新资产/费用
• 成本是"资金在资产内部的中间站"
• 成本类科目余额本身就是资产的一种形态（在产品）
成本是过程概念，不是结果概念：
• 资产、负债、权益：时点概念（月末余额）
• 收入、费用：期间概念（本月发生额）
• 成本：过渡概念（从资产到费用的中间形态）

## 分科目类别详解

### 1.资产类科目 → 借方余额正常
| 科目举例 | 借方余额含义 | 贷方余额含义       |
| ---- | ------ | ------------ |
| 银行存款 | 账户实有资金 | ❌ 透支（异常）     |
| 应收账款 | 客户欠款总额 | ❌ 多收款（异常）    |
| 库存商品 | 存货成本价值 | ❌ 库存红字（异常）   |
| 固定资产 | 资产原值   | ❌ 折旧超过原值（异常） |
逻辑：资产增加记借方，余额反映资产实有规模。

### 2.负债类科目 → 贷方余额正常
| 科目举例   | 贷方余额含义 | 借方余额含义     |
| ------ | ------ | ---------- |
| 应付账款   | 欠供应商货款 | ❌ 多付款（异常）  |
| 短期借款   | 银行借款余额 | ❌ 多还款（异常）  |
| 应付职工薪酬 | 应付工资总额 | ❌ 多付工资（异常） |
| 应交税费   | 应交未交税款 | ❌ 多缴税（异常）  |
逻辑：负债增加记贷方，余额反映债务规模。

### 3.所有者权益类 → 贷方余额正常
| 科目举例 | 贷方余额含义 | 借方余额含义     |
| ---- | ------ | ---------- |
| 实收资本 | 股东投入资本 | ❌ 抽逃资本（违法） |
| 本年利润 | 累计盈利   | 累计亏损（正常）   |
| 利润分配 | 未分配利润  | ❌ 超额分配（异常） |
特殊："本年利润"借方余额表示亏损，属于正常经营结果。

### 4.成本类科目 → 借方余额正常
| 科目举例 | 借方余额含义 | 贷方余额含义     |
| ---- | ------ | ---------- |
| 生产成本 | 在产品成本  | ❌ 结转错误（异常） |
| 制造费用 | 待分配费用  | ❌ 多分配（异常）  |
| 研发支出 | 资本化支出  | ❌ 冲销过度（异常） |
场景：月末"生产成本"借方余额即**在制品(WIP)**。

### 5.损益类科目 → 期末无余额
损益类科目月末需结转清零，余额方向由业务性质决定：
#### 5.1费用类（借方发生额）
| 科目举例 | 发生额方向        | 期末处理      |
| ---- | ------------ | --------- |
| 销售费用 | 借方发生         | 贷方结转→本年利润 |
| 管理费用 | 借方发生         | 贷方结转→本年利润 |
| 财务费用 | 借方发生（利息支出）   | 贷方结转→本年利润 |
| 利息收入 | 贷方发生（冲减财务费用） | 借方结转→本年利润 |
| 主营业务成本 | 借方发生     | 贷方结转→本年利润 |
| 其他业务成本 | 借方发生     | 贷方结转→本年利润 |
正常余额：结转前借方余额表示累计费用。

#### 5.2收入类（贷方发生额）
| 科目举例   | 发生额方向    | 期末处理      |
| ------ | -------- | --------- |
| 主营业务收入 | 贷方发生     | 借方结转→本年利润 |
| 其他业务收入 | 贷方发生     | 借方结转→本年利润 |
| 投资收益   | 贷方发生（收益） | 借方结转→本年利润 |
| 投资收益   | 借方发生（亏损） | 贷方结转→本年利润 |
正常余额：结转前贷方余额表示累计收入。
