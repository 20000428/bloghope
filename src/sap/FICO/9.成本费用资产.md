# 成本、费用、资产
<!-- more -->
## 资产处理

| 资产类型 | 会计处理 | SAP操作 | 凭证科目 | 原因 |
| --------- | ---- | ------ | ------ | ------ |
| ==有形资产-固定资产（设备、房屋）/使用权资产== | 折旧 | AFABN自动运行 | 累计折旧 | |
| ==无形资产（软件、专利）== | 摊销(类似折旧，叫法不同) | ABSO手动或转固后AFAB | 累计摊销 | 无实物形态 |
| ==在建工程== | 不折旧/摊销	 | 无 | 无 | 未完工，无法使用 |
| ==长期股权投资== | 不减值(除非做减值测试)	 | 无 | 无 | 非经营性资产 |
| ==商誉== | 不摊销，只做减值测试	 | 无 | 无 | 会计准则特殊规定 |

资产不一定都挂成本中心
| 固定资产用途     | 是否挂成本中心   | 成本中心类型    | 折旧去向           |
| ---------- | --------- | --------- | -------------- |
| **生产用设备**  | **必须挂** | 生产成本中心（F） | 制造费用→产品成本      |
| **管理用电脑**  | **必须挂** | 管理成本中心    | 管理费用（期间费用）     |
| **销售用车辆**  | **必须挂** | 销售成本中心    | 销售费用（期间费用）     |
| **研发用设备**  | **可选挂** | 研发内部订单    | 研发支出（费用/资本化）   |
| **闲置资产**   | **不挂**  | 无         | 管理费用-折旧费（期间费用） |
| **投资性房地产** | **不挂**  | 无         | 其他业务成本（非CO）    |
| **融资租入资产** | **不挂**  | 无         | 财务费用（使用权资产）    |


## 成本中心费用
成本中心的费用来源主要包括以下几大类：
1. 采购类
采购订单：材料、备件、办公用品、劳保用品等物资采购
费用性采购：服务采购（维修、咨询、运输等）
无PO发票：小额费用直接计入成本中心
2. 资产类
固定资产折旧：设备、房屋等折旧费用
无形资产摊销：软件、专利等摊销
低值易耗品摊销：未达到固定资产标准的资产
3. 人力资源成本
工资薪酬：通过工资核算模块过账
社保公积金：单位承担部分
职工福利费：福利、培训等费用
4. 费用报销类
员工报销：差旅费、交通费、通讯费、业务招待费等
信用卡记账：公务卡消费直接归集
5. 财务凭证类
手工记账凭证：水电费、租金、税费等直接记账
周期性凭证：如月度租金、保险费计提
6. 内部结算分摊
其他成本中心分摊：共同费用按比例分配
内部订单结算：项目费用结转到成本中心
作业类型重过账：调整错误的作业分配

成本归集流程
直接材料：
领料时直接计入工单（真正的直接计入）
直接人工：
第一步：工资费用先归集到人力资源成本中心
第二步：通过作业类型（如"人工工时"）和作业价格进行分配
第三步：报工时记录工时消耗量，系统自动计算：作业价格 × 报工工时 → 计入工单成本
制造费用：
归集在生产性成本中心 → 通过作业类型分配到工单
